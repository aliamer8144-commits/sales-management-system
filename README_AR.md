# نظام إدارة المبيعات

نظام مبيعات متكامل مبني بـ Next.js مع دعم اللغة العربية والتصميم المتجاوب للهواتف.

## المميزات

- 📱 تصميم متجاوب للهواتف (Mobile-First)
- 🌍 دعم كامل للغة العربية (RTL)
- ☁️ قاعدة بيانات سحابية (Turso) - مزامنة عبر جميع الأجهزة
- 👤 نظام صلاحيات (مدير / مستخدم)
- 📦 إدارة المنتجات مع تنبيهات المخزون
- 🧾 إنشاء فواتير (نقد / آجل)
- 👥 إدارة العملاء
- 📊 تقارير المبيعات والأرباح

## الأدوار

### المدير (Admin)
- إنشاء فواتير جديدة
- إدارة المنتجات (إضافة / تعديل / حذف)
- إدارة العملاء
- عرض جميع الفواتير
- عرض تقارير المبيعات والأرباح
- تنبيهات المخزون المنخفض

### المستخدم (User)
- إنشاء فواتير جديدة
- عرض مبيعاته الخاصة

## بيانات الدخول التجريبية

| الدور | البريد الإلكتروني | كلمة المرور |
|-------|-------------------|-------------|
| مدير | admin@example.com | admin123 |
| مستخدم | user@example.com | user123 |

## التقنيات المستخدمة

- **Frontend**: Next.js 16, React 19, TypeScript
- **Styling**: Tailwind CSS, shadcn/ui
- **Database**: Turso (libSQL)
- **Authentication**: Cookie-based

## التثبيت

```bash
# تثبيت الحزم
npm install

# تشغيل التطبيق
npm run dev
```
